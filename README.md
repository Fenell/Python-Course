# 🐍 Python Course - Bài học mở đầu

Repository này được xây dựng nhằm cung cấp các **bài học nhập môn Python** dành cho người mới bắt đầu.

Nội dung tập trung vào việc giúp người học làm quen với ngôn ngữ Python từ những kiến thức cơ bản nhất, thông qua các ví dụ và bài thực hành đơn giản.

---

## 📚 Nội dung chính

Các bài học mở đầu bao gồm:

- Giới thiệu về Python  
- Cách cài đặt và chạy chương trình Python  
- Biến và kiểu dữ liệu cơ bản  
- Toán tử trong Python  
- Câu lệnh điều kiện (`if/else`)  
- Vòng lặp (`for`, `while`)  
- Hàm (function) cơ bản  

---

## 🎯 Mục tiêu

- Giúp người học hiểu được cú pháp cơ bản của Python  
- Làm quen với tư duy lập trình  
- Tạo nền tảng để học các kiến thức nâng cao hơn  

---

## ▶️ Cách sử dụng

- Mở các file bài học trong repository  
- Đọc lý thuyết và chạy thử các ví dụ  
- Tự chỉnh sửa code để hiểu rõ hơn  

---

## 📌 Ghi chú

Project này phù hợp với:
- Người chưa từng học lập trình  
- Người muốn bắt đầu với Python một cách đơn giản, dễ hiểu
